# Email-Bot
Preston Smith's OSS email bot to sort through junk/spam emails. This bot is designed to be user-friendly and compatible with select email services: Gmail and Outlook Mail. This project is designed to be community-driven and open to anyone who wants to help remove annoying spam mail.

## Features
- Automatic junk email removal
- Possible Phishing alerts
- Toggle auto email reply with a preset message
- Sort emails by sender or keyphrase

## Usage
To use this project, simply clone this repository with your terminal

```
# clone the repository
git clone https://github.com/NAU-OSS/Email-Bot.git
# access the folder
cd Email-Bot
```
